# UI

## 1. 목적
- MVP 화면 구조를 단순하게 정의한다.

## 2. 구현 내용
- `/upload`
  - PDF/DOCX 업로드
  - 업로드된 파일 목록 표시
- `/chat`
  - 질문 입력
  - 답변 출력
  - retrieved sources 출력
- `/evaluation`
  - RAGAS 평균 지표
  - 질문별 결과 테이블

## 3. 현재 상태
- 미구현

## 4. 이슈 및 문제
- 실제 API 응답 구조가 아직 정해지지 않아 UI 상태 설계가 확정되지 않았다.
