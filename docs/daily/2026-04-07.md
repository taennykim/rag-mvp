- backend `/chat`은 아직 structured rewrite와 internal/external retrieval 분기 로직을 그대로 유지하고 있다.
- 따라서 현재 frontend와 backend 사이의 책임 경계는 임시 상태이며, 외부 RAG contract가 정해지면 adapter 기준으로 다시 맞춰야 한다.

- 현재 RAG 서버는 upload/chunk는 복구됐지만 Azure embedding firewall 이슈로 `indexed chunks`는 여전히 0이다.
