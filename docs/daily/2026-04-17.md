## 3. 이슈 및 문제
- 현재 frontend/backend 프로세스는 동작 중이지만, 화면 기능 검증은 실제 브라우저에서 항목별로 추가 확인이 필요하다.
- OpenAI 표준 endpoint처럼 `model` 필수를 요구하는 커스텀 서버라면 별도 고정 모델 처리 또는 adapter가 추가로 필요할 수 있다.
- RAG 서버 반영 시 경로를 잘못 지정하면 루트에 동일 파일명이 생성되고 실제 실행 코드가 갱신되지 않을 수 있어, 이후 동기화는 대상 경로를 명시적으로 검증해야 한다.
- backend runtime log에는 과거 `numpy._typing` 관련 ImportError 스택이 남아 있어 현재 이슈 재발 시 timestamp 기준으로 최신 로그와 구분해 확인해야 한다.
- Answer LLM 옵션에서 `Default (GPT-4o)`와 `GPT-4o`가 같은 deployment 값을 가리키므로 UX 관점에서 중복처럼 보일 수 있다(정책적으로 기본값 표기를 유지).
