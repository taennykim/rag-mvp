## 3. 이슈 및 문제
- frontend는 샌드박스에서 `3000` 포트 바인딩이 제한되어 권한 상승으로 실행했다.
- retrieval 질문 세트는 1차 초안이며, 실제 `/chat` hit 결과를 돌려본 뒤 fail 케이스 기준으로 보정이 필요하다.
- 현재 전역 retrieval은 산출방법서 계열 질문에서 오탐이 심하다.
- 파일 단독 retrieval는 가능하므로 indexing 누락보다 embedding/ranking 품질 문제 가능성이 높다.
- 단순 rerank로는 개선 폭이 제한적이어서 실제 embedding 모델 교체가 필요하다.
- `docs/retrieval.md`, `docs/plan.md`, `docs/backend.md` 일부 내용은 현재 실제 구현 상태보다 뒤쳐져 있다.
