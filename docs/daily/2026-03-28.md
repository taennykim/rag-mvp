## 4. 이슈 및 메모
- 현재 frontend/backend는 외부 공개 바인딩이 아니라 `127.0.0.1` 기준으로 운영 중이다.
- 따라서 화면 확인 시 직접 접속이 아니라 SSH port forwarding이 필요하다.
- parser 옵션은 현재 구현/검증용으로는 유용하지만, 사용자 기본 흐름에서는 자동 정책 고정이 더 적합할 가능성이 크다.
- 실제 retrieval 병목은 여전히 `hash embedding` 품질과 전역 ranking 품질에 가깝다.
